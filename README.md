# Nest
Nest es una solución a la organización académica la cual está diseñada para centralizar la vida escolar en un solo lugar. Esta aplicación combina herramientas de productividad personal con funciones colaborativas, todo bajo una estética moderna y dinámica.

## ¿Qué puedes hacer con Nest?
- Crear recordatorios
- Crear notas
- Crea grupos con tus amigos para compartir tareas, recordatorios, eventos, notas e incluso pueden crear encuestas para organizarse mejor
- Tener tu calendario a la mano y compartirlo con tus amigos
